# Project Vision

Xian Bai is a personal AI executive assistant that helps:

* Track work
* Manage projects
* Manage reminders
* Learn user preferences
* Analyze productivity
* Generate reviews
* Assist long-term goals

Principles:

* Assist, not spy
* Guide, not control
* Suggest, not force
